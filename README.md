# 🌐 Chisqui Web
# ================================

📌 Descripción

# ================================
# 🏡 Monte Los Olivos
# ================================

📌 Descripción

   Registro de personas miembros con generación automática de código QR individual.
    Creación y gestión de cultos (eventos).
    Escáner QR desde el navegador del celular para registrar asistencia en tiempo real.
    Dashboard con estadísticas de asistencia.
    Generación de reportes en PDF por culto.
    Soporte para acceso desde la red local (Wi-Fi) y exposición pública mediante ngrok.
    Panel de administración protegido por contraseña.

Tecnologías

Backend 	Python 3.11 · Flask · SQLAlchemy
Base de datos 	SQLite
QR 	qrcode + Pillow
Reportes 	ReportLab
Frontend 	HTML5 · CSS3 · JavaScript (vanilla)
Túnel público 	ngrok (opcional)

Repositorio: https://github.com/Jurgue12/Monte-Los-Olivos
